# Setting up server #

## Local development ##
* Install (Vagrant)[http://www.vagrantup.com]
* Install (VirtualBox)[https://www.virtualbox.org/]
* cd to here
* `vagrant up`
* `vagrant ssh`
* from within virtual box: `chmod 755 /vagrant/scripts/*`
* from within virtual box: `/vagrant/scripts/bootstrap-vm.sh`
